About the SkyHarvest Project

1. INTRODUCTION

SkyHarvest represents a revolutionary innovation in the agricultural sector, combining cutting-edge drone technology with digital monitoring and distribution platforms. Our drones can travel up to 400 kilometers (round trip) and carry up to 80 kilograms of cargo, in addition to their own weight, offering a complete solution for crop monitoring and direct-to-consumer delivery.

This technology allows farmers to access precise, real-time data about their crops, while consumers can select fresh produce directly from the source and track the entire harvesting and delivery process.

1.1 PROJECT OBJECTIVES

The main objective of SkyHarvest is to transform the agricultural supply chain, making it more efficient, sustainable, and transparent. We aim to:

  • Optimize crop monitoring through precise, real-time data
  • Reduce food waste through on-demand harvesting
  • Decrease the carbon footprint of agricultural product transportation
  • Directly connect producers and end consumers
  • Offer fresher products with complete traceability

1.2 PROJECT JUSTIFICATION

Modern agriculture faces significant challenges, including the need to increase productivity to feed a growing population while reducing environmental impact. SkyHarvest emerges as a response to these challenges, offering:

  • Precision: Detailed monitoring allows for precise interventions, reducing input usage
  • Efficiency: Elimination of intermediaries in the distribution chain
  • Sustainability: Electric drones reduce carbon emissions
  • Transparency: Consumers know the origin and journey of products
  • Accessibility: Technology available to small and medium producers

This solution is justified both by its potential to transform the agricultural sector and by the tangible benefits for producers, consumers, and the environment.

2. PROJECT DESCRIPTION

SkyHarvest consists of three main components:

  1. Drone Fleet: Equipped with multispectral sensors, high-resolution cameras, and autonomous navigation systems. Capable of carrying loads up to 80kg with a 400km range.
  2. Monitoring Platform: Cloud-based system that processes data collected by drones, generating insights about plant health, soil moisture, irrigation needs, and other agricultural parameters.
  3. Integrated Marketplace: Interface where consumers can view crops in real time, select desired products, and track the entire harvesting and delivery process.

The operation begins with regular crop monitoring by drones. The data is processed and made available to both farmers and consumers. When an order is placed, the drone is directed to precisely harvest the selected items, which are then transported directly to the customer's address with real-time tracking.

The system was developed with a focus on accessibility, allowing small and medium producers to access technology previously restricted to large agricultural operations. The interface was carefully designed to be intuitive for both farmers and end consumers.

2.2 IMPLEMENTATION DETAILS

Accessibility:

  • Controls to increase/decrease font size
  • High contrast toggle
  • Translation options (English/Spanish)
  • Semantic markup and ARIA where needed
  • Visible focus for keyboard navigation

Responsiveness:

  • Flexible layout using Bootstrap
  • Media queries for different screen sizes
  • Responsive images

Code Organization:

  • HTML and CSS separated in distinct folders
  • CSS divided into main and accessibility styles
  • Use of CSS variables for consistency

Visual Resources:

  • AI-generated images with documented prompts
  • Demonstration video
  • Icons for functionalities
  • Pleasant green-based color palette

Features without JavaScript:

  • All controls use CSS and pseudo-classes
  • Forms are properly marked up but without action
  • Simulated translation via CSS (show/hide)

3. FINAL CONSIDERATIONS

SkyHarvest represents a significant advancement at the intersection of agriculture and technology. By combining precise monitoring, on-demand harvesting, and efficient distribution, the project offers benefits for all links in the production chain.

Preliminary results from pilot farms demonstrate reductions of up to 30% in agricultural input usage, 15% increases in productivity, and 40% reductions in post-harvest waste. For consumers, the possibility of receiving fresh products directly from the farm, with complete transparency about their origin, represents a paradigm shift in food consumption.

The project continues to expand, with plans to incorporate new features such as predictive crop analysis and integration with automated irrigation systems. We believe SkyHarvest can become a standard in future agriculture, combining sustainability, efficiency, and transparency.
